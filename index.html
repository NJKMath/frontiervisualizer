About

This tool lets you visually lay out Battle Frontier sets into various groups, or analyze speed tiers in Speed Mode. There are three different modes, each with separate profiles:

Free Mode:

  • Create groups by Type, Move, Species
  • Use Advanced Filter for complex queries (e.g., "Type: Water AND Move: Surf")
  • Send groups to Speed Mode to help visualize speed tiers

Score Mode:

  • Assign numerical scores (0-100) to Pokemon via right-click or sidebar
  • Icons position vertically based on their scores
  • Focus mode allows detailed group management (up to 36 Pokemon)

Speed Mode:

  • Lays out all sets by calculated speed using current IV/Level settings
  • Speed value searches to highlight specific speeds, including boosts

All modes are completely separate, layouts for all three can be saved and loaded with JSON files.
